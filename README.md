# Smart Quran Search Engine (Hybrid AI) 🌙

محرك بحث قرآني متطور يدمج بين **البحث النصي الحرفي (Literal Search)** و **البحث الدلالي الذكي (Semantic Search)** باستخدام تقنيات الذكاء الاصطناعي.

## 🚀 المميزات
- **نظام بحث هجين (Hybrid Engine):** يبحث محلياً في ملف JSON للسرعة، ويستخدم Gemini API لفهم السياق والمعاني المعقدة.
- **دعم Gemini 2.5 Flash:** متوافق مع أحدث موديلات Google للذكاء الاصطناعي لعام 2026 لضمان سرعة استجابة فائقة.
- **خصوصية كاملة:** يتم تخزين مفتاح الـ API في الـ Local Storage الخاص بمتصفحك فقط.
- **واجهة مستخدم عصرية:** مصمم باستخدام Tailwind CSS مع دعم كامل للشاشات المختلفة (Responsive Design).
- **بدون قواعد بيانات معقدة:** يعمل مباشرة باستخدام ملفات JSON ثابتة، مما يجعله سهل النشر على GitHub Pages أو Vercel.

## 🛠 المتطلبات التقنية
- متصفح ويب حديث.
- مفتاح Google Gemini API.
- ملف `quran.json` يحتوي على نص المصحف الشريف بهيكلية (Surah -> Verses).

## 💻 التشغيل المحلي
1. قم بتحميل المستودع (Clone).
2. تأكد من وجود ملف `quran.json` في نفس المجلد.
3. قم بتشغيل خادم محلي (Local Server) لتجنب قيود CORS:
   ```bash
   python3 -m http.server 8000
افتح الرابط http://localhost:8000 في متصفحك.
